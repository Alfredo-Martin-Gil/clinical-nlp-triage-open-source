# Dyspnea — Triage Rules

IF any dyspnea Structured Red Flag present  
THEN high  
RATIONALE: Indicates possible respiratory failure or cardiac compromise.

IF no red flags AND ≥1 dyspnea Risk Factor present  
THEN intermediate  
RATIONALE: Risk factors increase likelihood of significant pathology.

IF no red flags AND no risk factors  
THEN low  
RATIONALE: Likely mild or self-limited cause.
