# Syncope — Triage Rules

IF any syncope Structured Red Flag present  
THEN high  
RATIONALE: High-risk features are associated with serious cardiac or neurological causes.

IF no red flags AND ≥1 syncope Risk Factor present  
THEN intermediate  
RATIONALE: Risk factors increase probability of underlying pathology.

IF no red flags AND no risk factors  
THEN low  
RATIONALE: Likely benign syncope pattern.
