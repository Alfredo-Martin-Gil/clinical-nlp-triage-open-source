# Stroke / Acute Neurological Deficit — Triage Rules

IF any stroke Structured Red Flag present  
THEN high  
RATIONALE: Time-sensitive neurological emergency.

IF no red flags AND ≥1 stroke Risk Factor present  
THEN intermediate  
RATIONALE: Increased probability of cerebrovascular pathology.

IF no red flags AND no risk factors  
THEN low  
RATIONALE: Lower likelihood of acute stroke.
